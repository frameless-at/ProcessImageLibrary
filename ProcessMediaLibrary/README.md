# ProcessMediaLibrary

ProcessWire admin module — a central table view of all images across all pages and image fields, with inline metadata editing.

See [MediaLibrary-Konzept.md](../MediaLibrary-Konzept.md) for the design document.

## Status

Phase 1 — work in progress. Module skeleton only.

## Requirements

- ProcessWire 3.0.172+
- PHP 8.0+

## Install

Copy the `ProcessMediaLibrary/` folder into `site/modules/`, then install via the ProcessWire admin (Modules → Refresh → Install „Media Library").

The module creates an admin page under Setup → Media Library and a `media-library-access` permission.
